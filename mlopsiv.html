MLOps Interview Questions

Below is a curated set of MLOps interview questions that span conceptual, scenario-based, and hands-on programming tasks. They are grouped so you can quickly pick the level or topic you need.

1. Conceptual & Process Questions

  • What is MLOps and how does it differ from traditional DevOps?
  • List the key components of an end-to-end MLOps pipeline.
  • Explain model versioning and why it is critical in production.
  • Describe the difference between model monitoring and model explainability; give two tools for each.
  • How do you ensure reproducibility across code, data, and models?
  • What strategies mitigate bias and fairness issues in an MLOps workflow?
  • Compare A/B testing vs Multi-Arm Bandit deployment strategies.

2. Scenario-Based (Troubleshooting & Design)

  • “Accuracy suddenly drops after deployment.” Outline your root-cause checklist.
  • “Data pipeline fails daily due to missing upstream fields.” How do you harden the pipeline?
  • “Cloud inference costs are exploding.” Propose three concrete cost-optimization levers.
  • “We need to roll back a bad model quickly.” Detail the rollback mechanism you would build.
  • “Multiple teams contribute to the same model.” How do you handle branching, merging, and artifact storage?

3. Programming & Hands-On Tasks

(Python is assumed; adapt as needed.)

Load & Split Data

Python

import pandas as pd
from sklearn.model_selection import train_test_split

def load_and_split(csv_path: str, test_size: float = 0.2):
    df = pd.read_csv(csv_path)
    X, y = df.drop('target', axis=1), df['target']
    return train_test_split(X, y, test_size=test_size, random_state=42)

Save / Load Model (persistence)

Python

import joblib

def save_model(model, path: str):
    joblib.dump(model, path)

def load_model(path: str):
    return joblib.load(path)

Basic REST API for Inference

Python

from flask import Flask, request, jsonify
import joblib

app = Flask(__name__)
model = joblib.load('model.pkl')

@app.route('/predict', methods=['POST'])
def predict():
    data = request.get_json()
    preds = model.predict([data['features']])
    return jsonify({'prediction': preds.tolist()})

Streaming Data-Drift Check (pseudo-code)

Use the Kolmogorov-Smirnov test in a Spark Streaming job:

Python

from scipy.stats import ks_2samp

def detect_drift(ref, new):
    stat, p = ks_2samp(ref, new)
    return p < 0.05   # boolean flag

Hyper-parameter Sweep Script

Python

from sklearn.model_selection import GridSearchCV

def tune(clf, X, y, param_grid, cv=5):
    gs = GridSearchCV(clf, param_grid, cv=cv)
    gs.fit(X, y)
    return gs.best_estimator_
